## 生成方法

1. 打开 [讯飞智作](https://www.xfzhizuo.cn/make)
2. 使用 `voice-all.mp3` 中的音色样本做声音定制
3. 按照 `voice-scripts-v2.md` 中的文案逐段生成
4. 语速设为偏慢，导出为MP3
5. 将导出的文件按上表命名，放入本目录

## 注意事项

- 文件名必须严格匹配，代码按文件名加载
- 格式：MP3（采样率不限，代码会自动解码为AudioBuffer）
- 如果某个文件缺失，对应步骤会静默跳过（不影响游戏流程）
